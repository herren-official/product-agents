# 세션 마무리 (wrap)

## 역할

현재 세션에서 수행한 작업을 정리하고, 다음 세션을 위한 인수인계를 완료한다.

## 트리거

- `/wrap`, "세션 마무리", "마무리해줘", "wrap up"

## 워크플로우

### 1. 학습 추출

이번 세션에서 발견한 재사용 가능한 지식을 식별:

- **피드백 메모리**: 사용자가 교정하거나 확인한 접근 방식
- **프로젝트 메모리**: 프로젝트 상태 변화, 진행 중인 작업 업데이트
- **컨벤션 발견**: 코드베이스에서 새로 파악한 패턴
- **도구 사용법**: MCP, 스킬, 에이전트 관련 학습

각 항목에 대해:
1. 기존 메모리(`memory/MEMORY.md`)에 이미 있는지 확인
2. 신규면 메모리 파일 생성 + MEMORY.md 인덱스 추가
3. 기존이면 업데이트 필요 여부 판단

### 2. CLAUDE.md 업데이트 제안

이번 세션에서 CLAUDE.md 또는 CLAUDE.local.md에 반영할 사항이 있는지 검토:

- 새로운 모듈/기술 스택 추가
- 명령어/워크플로우 변경
- 컨벤션 변경
- 주의사항 추가

> 직접 수정하지 않고 **제안만** 한다. 사용자 승인 후 반영.

### 3. 후속 작업 정리

#### 3-1. NEXT-SESSION-GUIDE 업데이트
- 현재 진행 상황 반영
- 다음 세션 작업 순서 명시
- 읽어야 할 파일 목록 업데이트
- 미완료 사항 체크리스트 갱신

#### 3-2. 미완료 작업 식별
- 진행 중이던 작업이 있으면 상태 기록
- 블로커/대기 사항 명시
- 우선순위 정리

### 4. 자동화 기회 탐지

이번 세션에서 반복적으로 수행한 작업 중 스킬/훅으로 자동화할 수 있는 것:

- 반복 패턴이 3회 이상이면 스킬화 제안
- 매번 수동으로 하는 검증이 있으면 훅 제안
- MCP 도구 설정이 필요하면 설정 제안

### 5. 임시 파일 정리

- `/tmp/` 하위 세션 임시 파일 삭제
- tool-results 캐시는 유지 (자동 관리)

## 출력 형식

```markdown
## 세션 마무리

### 학습 추출
- [신규/업데이트] {메모리 타입}: {내용 요약}

### CLAUDE.md 업데이트 제안
- {제안 내용} (승인 필요)

### 후속 작업
- NEXT-SESSION-GUIDE 업데이트: {완료/필요}
- 미완료: {목록}
- 블로커: {목록}

### 자동화 기회
- {제안}

### 임시 파일 정리
- {삭제된 파일 목록}
```
